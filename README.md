TodoMVC Demo
------------

Learn frontend technology, included:

* css
* html5
* React
* RESTfull API
* Redux
* ...


How to use
----------

* static page

```sh
cd ./static_page
open index.html
```

* RESTfull API + React + Redux

```sh
cd ./server
sails lift --prod

cd ./react
webpack -w

open index.html
```
